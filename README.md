# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using Python, PostgreSQL, SQL, and Power BI. The objective is to identify customer purchasing patterns, product performance, and business insights through data analysis and visualization.

## Dataset
- Customer Shopping Behavior Dataset
- 3,900 records
- 18 columns

## Tools & Technologies
- Python (Pandas, NumPy)
- PostgreSQL
- SQL
- Power BI
- GitHub

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. SQL Business Analysis
4. Power BI Dashboard
5. Business Recommendations

## Dashboard Features
- Revenue Analysis
- Customer Demographics
- Product Category Analysis
- Subscription Analysis
- Shipping Type Analysis

## Business Insights
- Identified top-performing product categories.
- Compared subscriber and non-subscriber spending.
- Analyzed customer purchase patterns.
- Provided business recommendations for growth.

## Project Files
- customer_shopping_behavior.ipynb
- customer_analysis.sql
- customer_behavior_dashboard.pbix
- customer_bahavior_analysis_report.docx
- Customer Shopping Behavior Analysis.pptx

## How to Run
1. Open the Jupyter Notebook.
2. Import the dataset.
3. Run the SQL queries in PostgreSQL.
4. Open the Power BI dashboard.
